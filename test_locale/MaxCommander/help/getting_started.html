Getting started with the MaxCommander app

Here you have a set of hints that will help you to start using the app.
  • You can use the MaxCommander app with your mouse, but if you want to be really effective, learn a few essential keyboard shortcuts and start using them on daily basis.
  • Operations like copying, moving and packing require a source and a destination. As the app have two panel, the currently active panel is always the source and the other one is the destination.
  • To switch the active panel, press Tab.
  • To select an item use Arrow keys on your keyboard (or click the item using the mouse).
  • To display the home folder press ~.
  • To display the next page of items, press ], for previous press [ and the last press Right arrow and for the first press Left arrow.
  • To show a folder content, select it and press Enter (or double click it with the mouse). To go back, press .
  • To display the same folder in the other panel, press Command + s.
  • To run an app or open a document in its editor app, select it and press Enter.
  • To preview a file with Quick Look, select it and press Space.
  • Command + 1 opens a list of bookmarks for the left panel, Command + 2 does the same for the right panel.
  • To start a file operation (like copying), select the target folder in one panel, then go to the other one, select items to be included in the operation and press Command + n (like Command + 5 for copying). See labels on the Button Bar for keyboard shortcuts.

Now go to the MaxCommander app and try all the keyboard shortcuts mentioned above!
