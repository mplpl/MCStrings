Using copy and paste for copying items

Using copy & paste for copying items is similar to drag and drop but you donâ€™t have to drag. You just select items and press Command + c in Finder and then Command + p in the MaxCommander app to past (copy items) into the current folder of the current panel in the app.

For the local file system items, this function also works in the other direction (selecting items in the MaxCommander with Command + c and paste into the Finder with Command + p) as well as between panels in the MaxCommander window (selecting items in the first panel in the MaxCommander, then Command + c, then change the panel with Tab, and Command + p).
