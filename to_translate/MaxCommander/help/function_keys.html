Using function keys

Traditional dual-panel file managers on other platform extensively use function keys to start their major file operations. Like most other computers, Macs have F1 – F12 function keys on their keyboards, but unlike others, most of them are assigned to specific functions. So that for example pressing F2 is not send to an app that is running but is interpreted on the operating system level. This behavior can be changed in System Preferences in Keyboard applet. By we DO NOT recommend this. Instead, all the function keys are mapped to Command + n combination in the MaxCommander app, when n is a function key number (for example, Command + 8 is the same as F8) – this works for all Macs and their keyboards without any reconfiguration required. And the buttons on the Button Bar are labels with that shortcuts.

Macs with keyboards equipped with the Touch Bar, can use function keys by pressing fn key. That make the Touch Bar display function keys that can then be tapped.
