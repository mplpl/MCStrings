Making folders

To create a new folder select a folder in which it should be created and press Command + 7. You will be asked for a new folder name. Entering it and clicking ‘OK’ button will start the operation.

When providing a folder name, you can also include entire new path to be created. For instance, typing “vacation/2021” will make “vacation” in the current folder and then “2021” in it.

Alternative invocation

You can also request this operation by pressing a button on the Button Bar or the Touch Bar or selecting an itme from the top menu.
