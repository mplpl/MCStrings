Deleting items

To delete a set of items, select them and press Command + 8 (or Backspace or Delete). After confirmation, a delete operation will start. The operation can be handled in background or with using multiple jobs. The latter is not allowed for deleting from archives.

If delete is requested for items on the local drives (not in an archive or remote file server) you can also decide if the items should be moved to a trash or deleted permanently. The default behavior can be set in “Settings” menu using “Move to Trash by Default” item.

The content of the other panel is not important for this operation.

Alternative invocation

You can also request this operation by pressing a button on the Button Bar or the Touch Bar or selecting an item from the top menu.
